## 简介

Text 组件用于创建各种文本元素，提供一致的排版样式和灵活的定制选项。该组件适用于需要统一文本样式的场景，如段落、提示文本、标签等。

组件特性：

- 支持多种文本大小选项
- 支持多种字体粗细选项
- 支持多种文本颜色选项
- 支持文本对齐方式
- 支持斜体和下划线样式
- 支持文本截断功能
- 支持自定义 HTML 元素类型
- 支持自定义 CSS 类名和内联样式
