## 简介

Alert 组件用于向用户显示重要的提示信息，支持多种类型的提示样式和交互效果。

组件特性：

- 支持四种预设类型，每种类型有对应的图标和颜色
- 可添加标题使提示更醒目
- 支持自定义 class 覆盖默认样式
- 响应式设计，适配不同屏幕尺寸
- 可控制是否显示图标
- 支持设置垂直方向外边距
- 支持多种样式变体
- 支持 v-model 双向绑定

## Props

### class

自定义 CSS 类名，用于覆盖默认样式。

<AlertVuePackage.CustomStyle />

### closable

是否可关闭，设置为 false 时隐藏关闭按钮。

<AlertVuePackage.Closable />

### description

描述文本，显示在标题下方，字体较小且透明度降低。

<AlertVuePackage.WithTitle />

### marginY

垂直方向外边距大小，支持预设的尺寸值。

<AlertVuePackage.MarginY />

### showIcon

是否显示图标，设置为 false 时隐藏类型对应的图标。

<AlertVuePackage.NoIcon />

### title

提示标题，可选，显示为粗体文本。

<AlertVuePackage.WithTitle />

### type

提示类型，影响颜色和图标，支持 info、success、warning、error 四种类型。

<AlertVuePackage.Types />

### variant

样式变体，支持 solid（实心）、outline（描边）、dash（虚线）、soft（柔和）四种风格。

<AlertVuePackage.Outline />

## Emits

### close

当点击关闭按钮时触发。

## Slots

### default

提示内容，支持任意文本内容。

### action

自定义操作按钮，显示在 Alert 右侧，适合放置自定义按钮或其他操作。

<AlertVuePackage.Action />
