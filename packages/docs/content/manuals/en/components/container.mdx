## Introduction

The Container component is a basic layout container used to limit content width and center display. It provides various sizes and padding options, suitable for different layout needs.

## Examples

<ComponentPackage.ComponentContainers.BasicUsage />

## Props

### aspectRatio

Aspect ratio (width/height), when set, the container will maintain this ratio, such as 16/9, 4/3, 1, etc.

<ComponentPackage.ComponentContainers.AspectRatioContainer />

### background

Background color type, supports all Tailwind background colors and opacity variants, such as: base-100, primary, secondary, accent, neutral, info, success, warning, error, and opacity variants like primary/10, secondary/20, etc.

<ComponentPackage.ComponentContainers.BackgroundContainer />

### border

Border size, optional values: none, sm, md, lg, xl.

<ComponentPackage.ComponentContainers.BorderContainer />

### centered

Whether to center display, when set to false, the container will be left-aligned.

### class

Custom CSS class name for adding additional styles.

<ComponentPackage.ComponentContainers.ClassContainer />

### contentCentered

Whether to center the internal content, when enabled, flex layout will be used to center the content within the container.

### fit

Content fit mode: none (default), contain (maintain aspect ratio, try to fill without overflow), cover (maintain aspect ratio, fill and may crop).

<ComponentPackage.ComponentContainers.FitContainer />

### flex

Flex layout direction, optional values: row, col, row-reverse, col-reverse. No flex layout when not set.

<ComponentPackage.ComponentContainers.FlexContainer />

### gap

Flex item spacing, optional values: none, xs, sm, md, lg, xl. Only effective when flex layout is enabled.

<ComponentPackage.ComponentContainers.GapContainer />

### height

Container height, optional values: none, xs, sm, md, lg, xl, 2xl, 3xl, 4xl, 5xl, 6xl, screen, auto. None means no height setting.

<ComponentPackage.ComponentContainers.HeightContainer />

### items

Flex item horizontal alignment, optional values: start, end, center, baseline, stretch. Only effective when flex layout is enabled.

### justify

Flex item vertical alignment, optional values: start, end, center, between, around, evenly. Only effective when flex layout is enabled.

### margin

Margin size, optional values: none, xs, sm, md, lg, xl, 2xl, 3xl, 4xl, 5xl.

<ComponentPackage.ComponentContainers.MarginContainer />

### muted

Whether to use muted style (inactive state). When set to true, the container will appear semi-transparent and disable interaction.

<ComponentPackage.ComponentContainers.MutedContainer />

### padding

Padding size, optional values: none, sm, md, lg, xl, 2xl, 3xl, 4xl.

<ComponentPackage.ComponentContainers.PaddingContainer />

### rounded

Rounded corner size, optional values: none, sm, md, lg, xl, full.

<ComponentPackage.ComponentContainers.RoundedContainer />

### shadow

Shadow size, optional values: none, sm, md, lg, xl, 2xl.

<ComponentPackage.ComponentContainers.ShadowContainer />

### width

Container width, optional values: xs, sm, md, lg, xl, full. Controls the maximum width of the container.

<ComponentPackage.ComponentContainers.WidthContainer />
