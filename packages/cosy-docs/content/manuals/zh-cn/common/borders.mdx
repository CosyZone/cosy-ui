## 简介

边框属性是 Cosy UI 组件库中的通用属性，用于设置组件的边框样式。支持多种边框尺寸，提供统一的视觉层次。

## 边框尺寸

### 可用选项
- `none` - 无边框
- `sm` - 细边框（1px）
- `md` - 中等边框（2px）
- `lg` - 粗边框（4px）
- `xl` - 超粗边框（8px）

<BorderShowcase />

## 使用方法

### 在组件中使用
```astro
<Container border="sm">
  细边框容器
</Container>

<Container border="md">
  中等边框容器
</Container>

<Container border="lg">
  粗边框容器
</Container>
```

### 在构建器模式中使用
```astro
const props = ContainerProps()
  .border("md")
  .build();

<Container {...props}>
  构建器模式设置边框
</Container>
```

## 相关组件

以下组件支持边框属性：
- `Container` - 容器组件
- `Card` - 卡片组件
- `Button` - 按钮组件
- `Alert` - 警告组件
- `Badge` - 徽章组件
- `Modal` - 模态框组件
