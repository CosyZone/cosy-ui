## 简介

Heading 组件用于创建各级标题，提供一致的排版样式和灵活的定制选项。该组件适用于需要清晰层次结构的内容页面，如文档、博客和产品介绍等。支持链接功能，当传入 `href` 属性时，标题会变成可点击的链接。

### align

<ComponentPackage.ComponentContainers.Background />

### class
通过 `class` 属性自定义标题样式。
<ComponentPackage.ComponentContainers.Class />

通过 `color` 属性自定义标题颜色。

### external

<ComponentPackage.ComponentContainers.External />
