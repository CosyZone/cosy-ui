# {frontmatter.title}

Image 组件是一个增强的图片组件，提供了丰富的功能，包括懒加载、加载状态指示、错误处理、以及各种视觉效果。

## alt
图片的替代文本，用于无障碍访问和图片加载失败时的显示。
<ComponentPackage.ComponentContainers.Basic />

## class
自定义 CSS 类名，用于覆盖默认样式。
<ComponentPackage.ComponentContainers.RoundedShadow />

## height
图片的高度，单位为像素。
<ComponentPackage.ComponentContainers.Basic />

## hover
图片的悬停效果，支持缩放、亮度调整和模糊效果。
<ComponentPackage.ComponentContainers.Hover />

## lazy
是否启用图片懒加载，默认为 true。
<ComponentPackage.ComponentContainers.Basic />

## loading
图片的加载方式，支持 lazy 和 eager。
<ComponentPackage.ComponentContainers.Basic />

## loadingIndicator
加载指示器类型，支持脉冲、旋转器、进度条和骨架屏。
<ComponentPackage.ComponentContainers.LoadingIndicatorSpinner />
<ComponentPackage.ComponentContainers.LoadingIndicatorProgress />

## objectFit
图片的填充方式，控制图片如何适应容器。
<ComponentPackage.ComponentContainers.ObjectFit />

## objectPosition
图片的位置，控制图片在容器中的对齐方式。
<ComponentPackage.ComponentContainers.ObjectFit />

## rounded
图片的圆角大小，提供多种圆角选项。
<ComponentPackage.ComponentContainers.RoundedShadow />

## shadow
图片的阴影效果，增强视觉层次感。
<ComponentPackage.ComponentContainers.RoundedShadow />

## showError
是否显示加载失败的错误图，默认为 true。
<ComponentPackage.ComponentContainers.Basic />

## showPlaceholder
是否显示加载中的占位图，默认为 true。
<ComponentPackage.ComponentContainers.LoadingIndicatorSpinner />

## src
图片源，可以是本地图片或远程URL。
<ComponentPackage.ComponentContainers.Basic />

## transition
图片的过渡动画效果，支持淡入、滑动和缩放。
<ComponentPackage.ComponentContainers.Transition />

## width
图片的宽度，单位为像素。
<ComponentPackage.ComponentContainers.Basic />
