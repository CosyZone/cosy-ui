## 简介

Vue 版本的 SmartIcon 组件是一个智能图标选择器，能够根据关键词自动渲染最合适的图标组件。它支持多个关键词，优先匹配第一个找到的图标，未匹配时渲染默认图标。该组件特别适用于需要根据内容动态选择图标的场景，如搜索结果显示、内容分类展示等。

## Vue 特有功能

### 响应式更新
Vue 版本的 SmartIcon 组件支持响应式更新，当 `keyword` 属性发生变化时，图标会自动重新渲染。

### 计算属性优化
使用 Vue 的计算属性来优化图标匹配逻辑，确保只有在关键词变化时才重新计算图标名称。

### 事件处理
组件支持 Vue 的事件系统，可以绑定 `@click`、`@mouseenter` 等事件处理器。

## 使用场景

### 搜索结果显示
在搜索功能中，根据搜索结果的内容类型自动显示相应的图标，提升用户体验。

### 内容分类展示
在内容管理系统中，根据内容的分类或标签自动选择对应的图标进行展示。

### 动态图标选择
在需要根据用户输入或系统状态动态选择图标的场景中，提供智能的图标匹配功能。

## 智能匹配逻辑

SmartIcon 采用多层次的匹配策略：

1. **直接匹配** - 优先查找完全匹配的关键词
2. **包含匹配** - 查找包含关键词的图标名称
3. **中文映射** - 支持中文关键词到图标的映射
4. **默认回退** - 未匹配时使用默认图标
