## 简介

Alert 组件用于向用户显示重要的提示信息，支持多种类型的提示样式和交互效果。

组件特性：

- 支持四种预设类型，每种类型有对应的图标和颜色
- 可添加标题使提示更醒目
- 支持自定义 class 覆盖默认样式
- 响应式设计，适配不同屏幕尺寸
- 可控制是否显示图标
- 支持设置垂直方向外边距
- 支持多种样式变体

## class

自定义 CSS 类名，用于覆盖默认样式。

<AlertPackage.CustomStyle />

## closable

是否可关闭，设置为 false 时隐藏关闭按钮。

<AlertPackage.Closable />

## description

描述文本，显示在标题下方，字体较小且透明度降低。

<AlertPackage.WithTitle />

## marginY

垂直方向外边距大小，支持预设的尺寸值。

<AlertPackage.MarginY />

## showIcon

是否显示图标，设置为 false 时隐藏类型对应的图标。

<AlertPackage.NoIcon />

## title

提示标题，可选，显示为粗体文本。

<AlertPackage.WithTitle />

## type

提示类型，影响颜色和图标，支持 info、success、warning、error 四种类型。

<AlertPackage.Types />

## variant

样式变体，支持 solid（实心）、outline（描边）、dash（虚线）、soft（柔和）四种风格。

### Outline 描边风格

通过 `variant="outline"` 可切换为描边风格：

<AlertPackage.Outline />

### Dash 虚线描边风格

通过 `variant="dash"` 可切换为虚线描边风格：

<AlertPackage.Dash />

### Soft 柔和风格

通过 `variant="soft"` 可切换为柔和风格：

<AlertPackage.Soft />

## 基础用法

最简单的提示信息展示：

<AlertPackage.Basic />

## 自定义操作按钮

通过 `action` slot 可以在 Alert 右侧自定义操作按钮：

<AlertPackage.Action />

该 slot 渲染在 Alert 右侧，适合放置自定义按钮或其他操作。

## Vue 组件

该组件也支持 Vue，用法如下：

<AlertPackage.Vue />
