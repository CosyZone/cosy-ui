# {frontmatter.title}

Text 组件用于创建各种文本元素，提供一致的排版样式和灵活的定制选项。该组件适用于需要统一文本样式的场景，如段落、提示文本、标签等。

## Props

### align
文本对齐方式。
<ComponentPackage.ComponentContainers.Align />

### as
渲染的 HTML 元素类型。
<ComponentPackage.ComponentContainers.As />

### class
自定义类名。
<ComponentPackage.ComponentContainers.CustomStyle />

### color
文本颜色。
<ComponentPackage.ComponentContainers.Color />

### size
文本大小。
<ComponentPackage.ComponentContainers.Size />

### truncate
是否截断超长文本。
<ComponentPackage.ComponentContainers.Truncate />

### weight
字体粗细。
<ComponentPackage.ComponentContainers.Weight />

### italic
是否使用斜体。
<ComponentPackage.ComponentContainers.Italic />

### underline
是否使用下划线。
<ComponentPackage.ComponentContainers.Underline />

### style
内联样式。
<ComponentPackage.ComponentContainers.Style />
