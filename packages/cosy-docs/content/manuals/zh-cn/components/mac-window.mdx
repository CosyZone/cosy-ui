## 简介

MacWindow 组件模拟 macOS 风格的应用窗口，包含标题栏、工具栏按钮、标签页和状态栏。适用于创建模拟桌面应用界面或代码编辑器等场景。

组件特性：

- 完整的 macOS 风格窗口外观，包含关闭、最小化、最大化按钮
- 支持标签页功能，可切换不同内容区域
- 提供工具栏和状态栏插槽，支持自定义内容
- 支持侧边栏布局，适合文件浏览器等应用
- 可自定义窗口高度和阴影效果
- 支持窗口操作事件回调
- 支持多种预设背景色主题，包括默认、主要、次要、强调、中性、信息、成功、警告、错误
