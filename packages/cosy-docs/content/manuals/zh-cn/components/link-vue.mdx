## 简介

Link 组件是一个功能强大的链接组件，不仅支持传统的链接样式，还提供了按钮风格、动画效果和丰富的自定义选项。通过语义化的属性配置，无需手动编写 CSS 类名即可实现各种视觉效果。

组件特性：

- 支持多种样式变体（default、primary、secondary、text、cta、ghost、light、navigation、github）
- 支持多种尺寸（sm、md、lg）
- 支持动画效果（hover-lift、hover-glow、hover-scale）
- 支持按钮风格和幽灵按钮
- 支持图标插槽（左侧、右侧、导航、GitHub）
- 支持外部链接自动新窗口打开
- 支持块级显示和全宽显示
- 响应式设计，适配不同屏幕尺寸

## Props

### active

是否为激活状态。

<LinkVuePackage.Active />

### align

对齐方式。可选值：`left`、`center`、`right`。

<LinkVuePackage.Align />

### animation

动画效果，定义鼠标悬停时的动画行为。可选值：`none`、`hover-lift`、`hover-glow`、`hover-scale`。

<LinkVuePackage.Animations />

### block

是否为块级显示，设置为 `true` 时链接会独占一行。

<LinkVuePackage.Block />

### btn

是否启用按钮风格，设置为 `true` 时会应用按钮样式。

<LinkVuePackage.ButtonStyle />

### circle

是否为圆形按钮，需要配合 `btn` 属性使用。

<LinkVuePackage.Circle />

### class

自定义 CSS 类名。

<LinkVuePackage.CustomClass />

### class:list

类名列表。

<LinkVuePackage.ClassList />

### color

按钮颜色，需要配合 `btn` 属性使用。可选值：`primary`、`secondary`、`accent`、`info`、`success`、`warning`、`error`。

<LinkVuePackage.Colors />

### debug

是否显示调试边框。

<LinkVuePackage.Debug />

### external

是否为外部链接，设置为 `true` 时会自动在新窗口打开。

<LinkVuePackage.External />

### fullWidth

是否占满宽度。

<LinkVuePackage.FullWidth />

### ghost

是否为幽灵按钮，需要配合 `btn` 属性使用。

<LinkVuePackage.Ghost />

### href

链接地址，必需属性。

<LinkVuePackage.Basic />

### icon

图标名称，支持所有可用的图标组件。

<LinkVuePackage.Icons />

### navigationType

导航类型，需要配合 `navigation` 变体使用。可选值：`previous`、`next`。

<LinkVuePackage.NavigationType />

### noUnderline

是否移除下划线，默认为 `true`。

<LinkVuePackage.NoUnderline />

### rounded

是否添加圆角。

<LinkVuePackage.Rounded />

### size

尺寸大小，控制链接的字体大小和间距。可选值：`sm`、`md`、`lg`。

<LinkVuePackage.Sizes />

### variant

样式变体，用于定义链接的视觉风格。可选值：`default`、`primary`、`secondary`、`text`、`cta`、`ghost`、`light`、`navigation`、`github`。

<LinkVuePackage.Variants />
