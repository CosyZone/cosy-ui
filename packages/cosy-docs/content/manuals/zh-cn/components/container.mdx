## 简介

Container 组件是一个基础的布局容器，用于限制内容宽度并居中显示。它提供了多种尺寸和内边距选项，适用于各种布局需求。

## 案例

<ContainerPackage.Basic />

## Props

### background
背景色类型，支持所有 Tailwind 背景色和透明度变体，如：base-100、primary、secondary、accent、neutral、info、success、warning、error，以及透明度变体如 primary/10、secondary/20 等。
<ContainerPackage.Background />

### border
是否显示边框，启用后会在容器周围添加边框和圆角。
<ContainerPackage.Border />

### centered
是否居中显示，设置为 false 时容器将靠左对齐。
<ContainerPackage.Centered />

### class
自定义 CSS 类名，用于添加额外的样式。
<ContainerPackage.CustomClass />

### class:list
类名列表，支持动态类名组合。
<ContainerPackage.ClassList />

### flex
flex 布局方向，可选值：row、col、row-reverse、col-reverse，不设置则不启用 flex 布局。
<ContainerPackage.Flex />

### gap
flex 项目间距，可选值：none、xs、sm、md、lg、xl，仅在启用 flex 布局时生效。
<ContainerPackage.Gap />

### items
flex 项目水平对齐方式，可选值：start、end、center、baseline、stretch，仅在启用 flex 布局时生效。
<ContainerPackage.Items />

### justify
flex 项目垂直对齐方式，可选值：start、end、center、between、around、evenly，仅在启用 flex 布局时生效。
<ContainerPackage.Justify />

### margin
外边距大小，可选值：none、xs、sm、md、lg、xl、2xl、3xl、4xl、5xl、6xl。
<ContainerPackage.Margin />

### padding
内边距大小，可选值：none、sm、md、lg、xl。
<ContainerPackage.Padding />

### rounded
圆角大小，可选值：none、sm、md、lg、xl、full。
<ContainerPackage.Rounded />

### size
容器尺寸，可选值：xs、sm、md、lg、xl、full，控制容器的最大宽度。
<ContainerPackage.Sizes />
