## 简介

Container 组件是一个基础的布局容器，用于限制内容宽度并居中显示。它提供了多种尺寸和内边距选项，适用于各种布局需求。

## 案例

<ComponentPackage.ComponentContainers.BasicUsage />

## Props

### aspectRatio

宽高比（宽/高），设置后容器会保持这个比例，如 16/9、4/3、1 等。

<ComponentPackage.ComponentContainers.AspectRatioContainer />

### background

背景色类型，支持所有 Tailwind 背景色和透明度变体，如：base-100、primary、secondary、accent、neutral、info、success、warning、error，以及透明度变体如 primary/10、secondary/20 等。

<ComponentPackage.ComponentContainers.BackgroundContainer />

### backgroundImage

背景图片源（本地 ImageMetadata 或 远程 URL）。提供时会用图片铺底作为背景。

<ComponentPackage.ComponentContainers.BackgroundImageContainer />

### border

边框尺寸，可选值：none、sm、md、lg、xl。

<ComponentPackage.ComponentContainers.BorderContainer />

### centered

是否居中显示，设置为 false 时容器将靠左对齐。

### class

自定义 CSS 类名，用于添加额外的样式。

<ComponentPackage.ComponentContainers.ClassContainer />

### contentCentered

是否让内部内容居中显示，启用后会在容器内使用 flex 布局居中内容。

### fit

内容适配模式：none（默认）、contain（保持比例，尽量占满且不溢出）、cover（保持比例，铺满并可能裁剪）。

<ComponentPackage.ComponentContainers.FitContainer />

### flex

flex 布局方向，可选值：row、col、row-reverse、col-reverse，不设置则不启用 flex 布局。

<ComponentPackage.ComponentContainers.FlexContainer />

### gap

flex 项目间距，可选值：none、xs、sm、md、lg、xl，仅在启用 flex 布局时生效。

<ComponentPackage.ComponentContainers.GapContainer />

### height

容器高度，可选值：none、xs、sm、md、lg、xl、2xl、3xl、4xl、5xl、6xl、screen、auto，none 表示不设置高度。

<ComponentPackage.ComponentContainers.HeightContainer />

### items

flex 项目水平对齐方式，可选值：start、end、center、baseline、stretch，仅在启用 flex 布局时生效。

### justify

flex 项目垂直对齐方式，可选值：start、end、center、between、around、evenly，仅在启用 flex 布局时生效。

### margin

外边距大小，可选值：none、xs、sm、md、lg、xl、2xl、3xl、4xl、5xl。

<ComponentPackage.ComponentContainers.MarginContainer />

### muted

是否使用柔和色样式（未激活状态），设置为 true 时容器会呈现半透明效果并禁用交互。

<ComponentPackage.ComponentContainers.MutedContainer />

### padding

内边距大小，可选值：none、sm、md、lg、xl、2xl、3xl、4xl。

<ComponentPackage.ComponentContainers.PaddingContainer />

### rounded

圆角大小，可选值：none、sm、md、lg、xl、full。

<ComponentPackage.ComponentContainers.RoundedContainer />

### shadow

阴影大小，可选值：none、sm、md、lg、xl、2xl。

<ComponentPackage.ComponentContainers.ShadowContainer />

### width

容器宽度，可选值：xs、sm、md、lg、xl、full，控制容器的最大宽度。

<ComponentPackage.ComponentContainers.WidthContainer />
