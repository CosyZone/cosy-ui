# {frontmatter.title}

用于制作图文绘本页。

- `PictureBookPage` 提供固定宽高比和按行等分的排版容器；
- `PictureBookTextBox` 以百分比定位，并可用“行数”定义高度，自动与父容器的行高对齐。

## 简介
适用于课程插画页、阅读绘本页、讲义封面等场景。
