# {frontmatter.title}

Login 组件用于创建灵活且功能丰富的登录页面，支持多种登录方式、自定义样式和布局。

## 基础用法

最基本的登录页面包含标题、用户名密码输入框、记住我选项、忘记密码链接和社交登录按钮。

<LoginPackage.LoginContainers.Basic />

## 自定义样式

你可以通过自定义类名、插槽等方式来自定义登录页面的样式和内容。

<LoginPackage.LoginContainers.CustomStyle />

## Props

### customClass

自定义 CSS 类名，用于覆盖默认样式。

<LoginPackage.LoginContainers.CustomClass />

### dividerText

社交登录分隔符文本，默认为"或"。

<LoginPackage.LoginContainers.DividerText />

### forgotPasswordText

忘记密码链接文本，默认为"忘记密码？"。

<LoginPackage.LoginContainers.ForgotPasswordText />

### loginButtonText

登录按钮文本，默认为"登录"。

<LoginPackage.LoginContainers.LoginButtonText />

### logo

登录页Logo图片路径。

<LoginPackage.LoginContainers.Logo />

### passwordLabel

密码输入框标签，默认为"密码"。

<LoginPackage.LoginContainers.PasswordLabel />

### rememberMeText

记住我选项文本，默认为"记住我"。

<LoginPackage.LoginContainers.RememberMeText />

### showForgotPassword

是否显示"忘记密码"链接，默认为 false。

<LoginPackage.LoginContainers.ShowForgotPassword />

### showRememberMe

是否显示"记住我"选项，默认为 false。

<LoginPackage.LoginContainers.ShowRememberMe />

### socialLogins

社交登录方式列表，支持 github、google、wechat。

<LoginPackage.LoginContainers.SocialLogins />

### subtitle

登录页副标题。

<LoginPackage.LoginContainers.Subtitle />

### title

登录页标题，默认为"登录"。

<LoginPackage.LoginContainers.Title />

### usernameLabel

用户名输入框标签，默认为"用户名"。

<LoginPackage.LoginContainers.UsernameLabel />

## Slots

### footer

通过 `footer` slot 可以在登录页面底部自定义内容：

<LoginPackage.LoginContainers.Footer />

### header

通过 `header` slot 可以在登录页面头部自定义内容：

<LoginPackage.LoginContainers.Header />

### socialButtons

通过 `socialButtons` slot 可以自定义社交登录按钮：

<LoginPackage.LoginContainers.SocialButtons />
