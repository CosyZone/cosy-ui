## 简介

ProductCard 组件用于展示产品信息，包括产品名称、图片、描述、App Store链接和产品官网链接。组件采用卡片式设计，支持悬停效果，并提供链接到产品相关页面的功能。

### 特性

- **多种尺寸** - 支持 xs、sm、md、lg、xl 五种预设尺寸
- **灵活布局** - 支持水平和垂直按钮布局
- **等高卡片** - 在网格布局中确保所有卡片高度一致
- **丰富链接** - 支持产品官网、App Store 和 GitHub 链接
- **自定义样式** - 支持自定义阴影、描述行数等样式

## Props

### appStoreUrl

App Store 链接地址，用于跳转到 App Store 应用页面。

<ProductCardPackage.ComponentContainers.AppStoreUrl />

### background

卡片背景色，支持基础颜色、透明度变体和渐变背景。

<ProductCardPackage.ComponentContainers.Background />

### buttonLayout

按钮布局方向，支持水平（row）和垂直（column）两种布局方式。

<ProductCardPackage.ComponentContainers.ButtonLayout />

### class

自定义 CSS 类名，用于覆盖默认样式。

<ProductCardPackage.ComponentContainers.Class />

### description

产品描述文本，支持多行显示。

<ProductCardPackage.ComponentContainers.Description />

### descriptionLines

<ProductCardPackage.ComponentContainers.DescriptionLines />

### equalHeight

是否启用等高卡片，在网格布局中确保所有卡片高度一致。

<ProductCardPackage.ComponentContainers.EqualHeight />

### githubButtonText

GitHub 按钮的显示文本。

<ProductCardPackage.ComponentContainers.GithubButtonText />

### githubUrl

<ProductCardPackage.ComponentContainers.GithubUrl />

<ProductCardPackage.ComponentContainers.Image />

### muted

### secondaryButtonText
