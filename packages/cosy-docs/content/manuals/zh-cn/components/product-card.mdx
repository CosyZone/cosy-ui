## 简介

ProductCard 组件用于展示产品信息，包括产品名称、图片、描述、App Store链接和产品官网链接。组件采用卡片式设计，支持悬停效果，并提供链接到产品相关页面的功能。

## 案例

<ProductCardPackage.ComponentContainers.Basic />

## Props

### appStoreUrl
App Store 链接地址，用于跳转到 App Store 应用页面。
<ProductCardPackage.ComponentContainers.Basic />

### buttonLayout
按钮布局方向，支持水平（row）和垂直（column）两种布局方式。
<ProductCardPackage.ComponentContainers.Basic />

### class
自定义 CSS 类名，用于覆盖默认样式。
<ProductCardPackage.ComponentContainers.Basic />

### description
产品描述文本，支持多行显示。
<ProductCardPackage.ComponentContainers.Basic />

### descriptionLines
描述文本的最大显示行数，超出部分会被截断。
<ProductCardPackage.ComponentContainers.DescriptionLines />

### equalHeight
是否启用等高卡片，在网格布局中确保所有卡片高度一致。
<ProductCardPackage.ComponentContainers.EqualHeight />

### githubButtonText
GitHub 按钮的显示文本。
<ProductCardPackage.ComponentContainers.Basic />

### githubUrl
GitHub 仓库链接地址。
<ProductCardPackage.ComponentContainers.Basic />

### image
产品图片，支持多种图片源格式。
<ProductCardPackage.ComponentContainers.Basic />

### name
产品名称，必填属性。
<ProductCardPackage.ComponentContainers.Basic />

### primaryButtonText
主按钮（产品官网按钮）的显示文本。
<ProductCardPackage.ComponentContainers.Basic />

### productUrl
产品官网链接地址。
<ProductCardPackage.ComponentContainers.Basic />

### secondaryButtonText
次按钮（App Store 按钮）的显示文本。
<ProductCardPackage.ComponentContainers.Basic />

### shadow
卡片阴影样式，提供 none、sm、md、lg、xl 五种样式选择。
<ProductCardPackage.ComponentContainers.Shadow />

### size
卡片尺寸，支持 xs、sm、md、lg、xl 五种预设尺寸。
<ProductCardPackage.ComponentContainers.Sizes />
