# Icons Vue

## 简介

Icons Vue 组件库提供了一套完整的 Vue 图标组件，基于 SVG 图标构建，支持自定义大小、颜色和样式。所有图标都经过精心设计，确保在不同尺寸下都能保持清晰和美观。

## class

自定义 CSS 类名，用于覆盖默认样式。

<IconsVuePackage.IconShowcase />

## color

图标的颜色，支持任何有效的 CSS 颜色值。

<IconsVuePackage.IconColors />

## size

图标的大小，支持任何有效的 CSS 尺寸值。

<IconsVuePackage.IconSizes />

## 使用方法

### 基础用法

```vue
<template>
  <UserIcon />
  <SettingsIcon size="32px" color="blue" />
  <SearchIcon class="custom-icon" />
</template>

<script setup>
import { UserIcon, SettingsIcon, SearchIcon } from '@coffic/cosy-ui/vue';
</script>
```

### 自定义样式

```vue
<template>
  <div>
    <HeartIcon 
      size="48px" 
      color="#ef4444" 
      class="cosy:animate-pulse" 
    />
  </div>
</template>

<script setup>
import { HeartIcon } from '@coffic/cosy-ui/vue';
</script>
```

## 技术特性

- **TypeScript 支持** - 完整的类型定义
- **响应式设计** - 支持不同屏幕尺寸
- **可访问性** - 符合 WCAG 标准
- **性能优化** - 轻量级 SVG 图标
- **主题支持** - 支持深色/浅色主题切换

## 注意事项

1. 所有图标都基于 24x24 的网格设计，确保视觉一致性
2. 图标颜色默认为 `currentColor`，会自动继承父元素的文本颜色
3. 图标大小支持任何有效的 CSS 尺寸值，包括像素、em、rem 等
4. 自定义类名可以通过 `class` 属性传入，支持 Tailwind CSS 等框架
