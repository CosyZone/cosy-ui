## 简介

ImageRow 组件用于展示一行图片，基于 Container 组件构建，支持丰富的背景配置。专门用于水平排列的图片展示场景，支持丰富的样式效果和响应式设计。

<ImageRowPackage.Case />

## Props

### arrowVariant

箭头按钮样式变体，可选值：ghost、outline、primary、secondary。

### background

背景色类型，支持所有 Tailwind 背景色和透明度变体。

<ImageRowPackage.Background />

### border

是否显示边框，设置为 true 时会在容器周围添加边框。

<ImageRowPackage.Border />

### class

自定义 CSS 类名，用于添加额外的样式。

### containerRounded

容器圆角，可选值：none、sm、md、lg、xl、2xl、3xl、full。

<ImageRowPackage.ContainerRounded />

### gap

图片间距，可选值：none、xs、sm、md、lg、xl。

<ImageRowPackage.Gap />

### hover

悬停效果，可选值：none、scale、brightness、blur。

<ImageRowPackage.Hover />

### images

图片数组，每个图片项包含源地址、替代文本等信息。

<ImageRowPackage.Images />

### margin

外边距大小，可选值：none、xs、sm、md、lg、xl、2xl、3xl、4xl、5xl、6xl。

<ImageRowPackage.Margin />

### padding

内边距大小，可选值：none、sm、md、lg、xl、2xl、3xl、4xl。

<ImageRowPackage.Padding />

### rounded

图片圆角，可选值：none、sm、md、lg、xl、2xl、3xl、full。

<ImageRowPackage.Rounded />

### shadow

图片阴影，可选值：none、sm、md、lg、xl、2xl。

<ImageRowPackage.Shadow />

### showArrows

是否显示左右导航箭头，设置为 true 时会在图片行两侧显示滚动箭头。

<ImageRowPackage.Arrows />

### width

容器宽度，可选值：xs、sm、md、lg、xl、full。

<ImageRowPackage.Width />
