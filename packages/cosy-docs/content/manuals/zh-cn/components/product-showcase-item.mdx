## 简介

ProductShowcaseItem 组件用于展示单个产品的详细信息，采用苹果风格的大图展示设计。支持左右交替布局、新品标签、价格显示和操作按钮。适用于产品详情页、产品展示页面等场景。

组件特性：

- 大图展示 - 突出产品图片，采用高质量大图展示
- 交替布局 - 支持左右交替布局，增加视觉层次
- 信息层次 - 清晰的信息层次：标签、标题、描述、价格、按钮
- 苹果风格 - 采用苹果官网的简洁、优雅设计风格
- 响应式设计 - 在不同设备上都能良好展示

## 使用场景

- 单个产品详情展示
- 产品展示页面中的单个产品
- 苹果风格的产品介绍
- 产品对比页面
- 特色产品突出展示

## 设计理念

ProductShowcaseItem 组件采用苹果官网的设计理念：

1. **大图展示** - 突出产品图片，让用户直观了解产品外观
2. **信息层次** - 清晰的信息层次：新品标签 → 产品标题 → 描述 → 价格 → 操作按钮
3. **简洁优雅** - 采用极简设计，突出产品本身
4. **交互友好** - 提供明确的操作按钮，引导用户行为
5. **视觉平衡** - 通过左右布局实现视觉平衡

## 最佳实践

1. **图片质量** - 使用高质量的产品图片，建议尺寸为 800x600 或更高
2. **描述简洁** - 产品描述应简洁明了，突出核心卖点
3. **价格清晰** - 价格显示应清晰醒目，使用合适的字体大小
4. **按钮文案** - 按钮文案应明确表达操作意图
5. **新品标签** - 合理使用新品标签，突出新产品

## 布局说明

组件采用两列网格布局：

- **左列**：产品信息（标签、标题、描述、价格、按钮）
- **右列**：产品图片

当 `align="right"` 时，布局会反转：
- **左列**：产品图片
- **右列**：产品信息

## 注意事项

- 图片建议使用高分辨率，以确保在大图展示时效果良好
- 组件内部使用 Container 组件，确保内容不会过宽
- 背景色会应用到整个组件区域
- 按钮点击事件需要自行处理，组件只提供回调接口
