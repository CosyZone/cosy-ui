## 简介

Link 组件是一个功能强大的链接组件，不仅支持传统的链接样式，还提供了按钮风格、动画效果和丰富的自定义选项。通过语义化的属性配置，无需手动编写 CSS 类名即可实现各种视觉效果。

<LinkPackage.Basic />

## Props

### active

是否为激活状态。

<LinkPackage.Active />

### align

对齐方式。可选值：`left`、`center`、`right`。

<LinkPackage.Align />

### animation

动画效果，定义鼠标悬停时的动画行为。可选值：`none`、`hover-lift`、`hover-glow`、`hover-scale`。

<LinkPackage.Animations />

### block

是否为块级显示，设置为 `true` 时链接会独占一行。

<LinkPackage.Block />

### btn

是否启用按钮风格，设置为 `true` 时会应用按钮样式。

<LinkPackage.ButtonStyle />

### circle

是否为圆形按钮，需要配合 `btn` 属性使用。

<LinkPackage.Circle />

### class

自定义 CSS 类名。

<LinkPackage.CustomClass />

### class:list

类名列表。

<LinkPackage.ClassList />

### color

按钮颜色，需要配合 `btn` 属性使用。可选值：`primary`、`secondary`、`accent`、`info`、`success`、`warning`、`error`。

<LinkPackage.Colors />

### debug

是否显示调试边框。

<LinkPackage.Debug />

### external

是否为外部链接，设置为 `true` 时会自动在新窗口打开。

<LinkPackage.External />

### fullWidth

是否占满宽度。

<LinkPackage.FullWidth />

### ghost

是否为幽灵按钮，需要配合 `btn` 属性使用。

<LinkPackage.Ghost />

### href

链接地址，必需属性。

<LinkPackage.Basic />

### icon

图标名称，支持所有可用的图标组件。

<LinkPackage.Icons />

### navigationType

导航类型，需要配合 `navigation` 变体使用。可选值：`previous`、`next`。

<LinkPackage.NavigationType />

### noUnderline

是否移除下划线，默认为 `true`。

<LinkPackage.NoUnderline />

### rounded

是否添加圆角。

<LinkPackage.Rounded />

### size

尺寸大小，控制链接的字体大小和间距。可选值：`sm`、`md`、`lg`。

<LinkPackage.Sizes />

### variant

样式变体，用于定义链接的视觉风格。可选值：`default`、`primary`、`secondary`、`text`、`cta`、`ghost`、`light`、`navigation`、`github`。

<LinkPackage.Variants />

### rest

其他 HTML `<a>` 标签属性。

<LinkPackage.Rest />

## Slots

### default

链接的文本内容。

<LinkPackage.Basic />
