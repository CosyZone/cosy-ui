## 简介

Link 组件是一个功能强大的链接组件，不仅支持传统的链接样式，还提供了按钮风格、动画效果和丰富的自定义选项。通过语义化的属性配置，无需手动编写 CSS 类名即可实现各种视觉效果。

**主要特性：**
- 🎨 多种样式变体（default、primary、secondary、text、cta、ghost、light、github）
- 🔘 完整的按钮风格支持（普通、幽灵、圆形、各种尺寸）
- ✨ 丰富的动画效果（上浮、发光、缩放）
- 📱 响应式设计，支持各种对齐方式
- 🎯 语义化属性，无需手动拼接 CSS 类
- ♿ 完善的无障碍支持
- 🔗 自动图标支持（GitHub 链接自动显示 GitHub 图标）

最简单的链接使用方式，支持内部链接和外部链接。

## 样式变体

通过 `variant` 属性应用不同的样式变体，适用于不同的使用场景。

<LinkPackage.ComponentContainers.Variants />

## GitHub 链接

使用 `variant="github"` 创建带有 GitHub 图标的链接，自动在链接前显示 GitHub 图标。

<LinkPackage.ComponentContainers.Github />

## 文档导航

使用 `variant="navigation"` 创建专门用于文档中"上一个"、"下一个"导航的链接样式。

<LinkPackage.ComponentContainers.Navigation />

## API 参考

### Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| `href` | 链接地址 | `string` | - |
| `external` | 是否为外部链接，自动新窗口打开 | `boolean` | `false` |
| `variant` | 样式变体 | `'default' \| 'primary' \| 'secondary' \| 'text' \| 'cta' \| 'ghost' \| 'light' \| 'navigation' \| 'github'` | `'default'` |
| `size` | 尺寸大小 | `'sm' \| 'md' \| 'lg'` | `'md'` |
| `animation` | 动画效果 | `'none' \| 'hover-lift' \| 'hover-glow' \| 'hover-scale'` | `'none'` |
| `block` | 是否为块级显示 | `boolean` | `false` |
| `btn` | 是否启用按钮风格 | `boolean` | `false` |
| `circle` | 是否为圆形按钮（需配合 `btn` 使用） | `boolean` | `false` |
| `ghost` | 是否为幽灵按钮（需配合 `btn` 使用） | `boolean` | `false` |
| `color` | 按钮颜色（需配合 `btn` 使用） | `'primary' \| 'secondary' \| 'accent' \| 'info' \| 'success' \| 'warning' \| 'error'` | - |
| `align` | 对齐方式 | `'left' \| 'center' \| 'right'` | - |
| `rounded` | 是否添加圆角 | `boolean` | `false` |
| `noUnderline` | 是否移除下划线 | `boolean` | `false` |
| `fullWidth` | 是否占满宽度 | `boolean` | `false` |
| `centerText` | 是否居中文本（兼容属性） | `boolean` | `false` |
| `debug` | 是否显示调试边框 | `boolean` | `false` |
| `navigationType` | 导航类型，自动添加相应图标 | `'previous' \| 'next'` | - |
| `class` | 自定义 CSS 类名 | `string` | - |
| `...rest` | 其他 HTML `<a>` 标签属性 | `HTMLAttributes<'a'>` | - |
