## 简介

Link 组件是一个功能强大的链接组件，不仅支持传统的链接样式，还提供了按钮风格、动画效果和丰富的自定义选项。通过语义化的属性配置，无需手动编写 CSS 类名即可实现各种视觉效果。

## 基础用法

最简单的链接使用方式，支持内部链接和外部链接。

<LinkPackage.ComponentContainers.Basic />

## 样式变体

通过 `variant` 属性应用不同的样式变体，适用于不同的使用场景。包括 GitHub 链接变体和文档导航变体。

<LinkPackage.ComponentContainers.Variants />

## 图标功能

通过 `icon` 属性可以为链接添加各种图标。

<LinkPackage.ComponentContainers.Icons />

## 按钮风格

通过 `btn` 属性启用按钮风格，支持多种颜色、尺寸和形状。

<LinkPackage.ComponentContainers.ButtonStyle />

## 动画效果

通过 `animation` 属性添加悬停动画效果，提升用户体验。

<LinkPackage.ComponentContainers.Animations />

## 高级功能

更多高级功能，包括对齐方式、样式修饰、调试模式等。
