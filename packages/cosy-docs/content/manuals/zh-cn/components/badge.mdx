## 简介

Badge 组件用于突出显示少量信息，如标签、状态或计数。

组件特性：

- 支持多种预设颜色
- 支持多种尺寸
- 支持描边样式
- 响应式设计，适配不同屏幕尺寸

## 案例

<BadgePackage.Basic />

## Props

### class

自定义 CSS 类名，用于覆盖默认样式。

<BadgePackage.Class />

### outline

徽章是否为描边样式，设置为 true 时显示为描边风格。

<BadgePackage.Outline />

### size

徽章的尺寸，支持 xs、sm、md、lg 四种尺寸。

<BadgePackage.Sizes />

### variant

徽章的颜色变体，支持多种预设颜色主题。

<BadgePackage.Variant />

## Slots

### default

徽章内容，支持任意文本内容。
