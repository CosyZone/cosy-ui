## 简介

Images 组件用于展示一组图片，支持多种布局方式和丰富的样式效果。可以像你展示的那样美观地展示图片集合，支持网格、瀑布流、轮播和堆叠等多种布局。

<ImagesPackage.Case />

## Props

### class

自定义 CSS 类名，用于添加额外的样式。

### columns

网格布局的列数，当 layout 为 grid 时有效。

<ImagesPackage.Columns />

### gap

图片间距，可选值：none、xs、sm、md、lg、xl。

<ImagesPackage.Gap />

### hover

悬停效果，可选值：none、scale、brightness、blur。

<ImagesPackage.Hover />

### images

图片数组，每个图片项包含源地址、替代文本等信息。

<ImagesPackage.Images />

### layout

布局方式，可选值：grid（网格）、masonry（瀑布流）、carousel（轮播）、stack（堆叠）。

<ImagesPackage.Layouts />

### rounded

图片圆角，可选值：none、sm、md、lg、xl、2xl、3xl、full。

<ImagesPackage.Rounded />

### shadow

图片阴影，可选值：none、sm、md、lg、xl、2xl。

<ImagesPackage.Shadow />
