## 简介

Grid 组件是一个灵活的网格布局组件，用于创建响应式的多列布局。它封装了 CSS Grid 的常用功能，提供简单易用的接口来控制网格的列数和间距。

组件特性：

- 支持固定列数和响应式列数配置
- 提供多种间距选项（行间距、列间距、整体间距）
- 支持垂直外边距设置
- 可选的边框显示
- 响应式设计，适配不同屏幕尺寸
