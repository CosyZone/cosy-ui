## 简介

ProductHero 组件用于展示单个产品的详细信息，采用苹果风格的大图展示设计。支持左右交替布局、标签数组、价格显示和操作按钮。适用于产品详情页、产品展示页面等场景。

组件特性：

- 大图展示 - 突出产品图片，采用高质量大图展示
- 交替布局 - 支持左右交替布局，增加视觉层次
- 信息层次 - 清晰的信息层次：标签、标题、描述、价格、按钮
- 苹果风格 - 采用苹果官网的简洁、优雅设计风格
- 响应式设计 - 在不同设备上都能良好展示

## Props

### align

布局对齐方式，支持 `left`（左对齐，默认）和 `right`（右对齐）。

<ProductHeroPackage.Align />

### background

背景色类型，使用预定义的背景色类型，如 'primary', 'base-200', 'gradient-sky' 等，默认为 'base-100'。

<ProductHeroPackage.Background />

### class

自定义CSS类，可用于覆盖默认样式。

<ProductHeroPackage.Class />

### description

产品描述，必填属性。

<ProductHeroPackage.Description />

### image

产品图片，必填属性。支持 ImageSource 类型。

<ProductHeroPackage.Image />

### labels

标签数组，传入一个或多个字符串作为标签。

<ProductHeroPackage.Labels />

### name

产品名称，必填属性。

<ProductHeroPackage.Name />

### highlight

高亮文案（如价格、宣传语等），可选属性；未提供时隐藏该行。

<ProductHeroPackage.Highlight />

## Slots

### actions

自定义操作区内容（如按钮组），渲染在描述与图片下方。

<ProductHeroPackage.Actions />
