## 简介

SocialIcon Vue 组件用于显示社交媒体平台的图标，支持多种常见的社交媒体平台。它提供了一致的样式和交互效果，同时支持自定义大小和颜色。

## Props

### platform

社交媒体平台名称，支持以下平台：

- `github` - GitHub 平台
- `twitter` - Twitter 平台  
- `linkedin` - LinkedIn 平台
- `facebook` - Facebook 平台

<SocialIcon platform="github" size="32px" color="#333" />

### size

图标的大小，支持任何有效的 CSS 尺寸值。

<div style="display: flex; gap: 1rem; align-items: center;">
  <SocialIcon platform="github" size="16px" color="#333" />
  <SocialIcon platform="github" size="24px" color="#333" />
  <SocialIcon platform="github" size="32px" color="#333" />
  <SocialIcon platform="github" size="48px" color="#333" />
</div>

### color

图标的颜色，支持任何有效的 CSS 颜色值。

<div style="display: flex; gap: 1rem; align-items: center;">
  <SocialIcon platform="github" size="24px" color="#333" />
  <SocialIcon platform="github" size="24px" color="#0366d6" />
  <SocialIcon platform="github" size="24px" color="#28a745" />
  <SocialIcon platform="github" size="24px" color="#d73a49" />
</div>

### class

自定义 CSS 类名，用于覆盖默认样式。

<SocialIcon platform="github" size="32px" color="#333" class="custom-social-icon" />

## 使用场景

- 社交媒体链接按钮
- 团队成员的社交资料展示
- 网站的社交分享功能
- 应用内的社交登录选项

## 设计理念

1. **简洁明了** - 使用清晰易识别的图标表示社交媒体平台
2. **一致性** - 保持统一的样式和交互效果
3. **可定制性** - 支持自定义大小、颜色和样式
4. **可访问性** - 提供适当的 aria 标签，确保屏幕阅读器可以识别

## Vue 特性

- 使用 Vue 3 Composition API
- 响应式属性绑定
- 计算属性优化性能
- TypeScript 类型支持
