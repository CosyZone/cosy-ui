# {frontmatter.title}

CardBook 组件用于展示一本书的信息。支持传入封面图；当未提供封面时，将根据书名使用智能渐变背景作为占位，并在中央显示书名。

## 简介
适用于课程、图书列表、阅读推荐等场景。
