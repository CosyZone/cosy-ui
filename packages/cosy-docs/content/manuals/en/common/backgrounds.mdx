## Introduction

Background color properties are common properties in the Cosy UI component library, used to set the background color of components. Supports basic background colors, opacity variants, and gradient colors.

## Basic Background Colors

### Theme Colors
- `primary` - Primary color
- `secondary` - Secondary color
- `accent` - Accent color
- `neutral` - Neutral color

<ThemeColorsShowcaseEn />

### Base Colors
- `base-100` - Base background color (light)
- `base-200` - Base background color (medium)
- `base-300` - Base background color (dark)

### Status Colors
- `success` - Success color
- `warning` - Warning color
- `error` - Error color
- `info` - Info color

### Special Values
- `transparent` - Transparent background
- `none` - No background

## Opacity Variants

Each basic background color supports opacity variants in the format `color/opacity`:

### Opacity Values
- `10` - 10% opacity
- `20` - 20% opacity
- `30` - 30% opacity
- `40` - 40% opacity
- `50` - 50% opacity
- `60` - 60% opacity
- `70` - 70% opacity
- `80` - 80% opacity
- `90` - 90% opacity

### Examples
- `primary/20` - Primary color with 20% opacity
- `success/10` - Success color with 10% opacity
- `error/30` - Error color with 30% opacity

## Usage

### In Components
```astro
<Container background="primary">
  Primary background
</Container>

<Container background="success/20">
  Success background with 20% opacity
</Container>

<Container background="gradient-sky">
  Sky gradient background
</Container>
```

### In Builder Pattern
```astro
const props = ContainerProps()
  .background("primary/20")
  .build();

<Container {...props}>
  Builder pattern background
</Container>
```

## Best Practices

### 1. Choose Appropriate Background Colors
- **Main Content**: Use `base-100` or `base-200`
- **Emphasized Content**: Use `primary` or `accent`
- **Status Indicators**: Use corresponding status colors (`success`, `warning`, `error`, `info`)

### 2. Opacity Usage
- **Light Emphasis**: Use `10-20%` opacity
- **Medium Emphasis**: Use `30-50%` opacity
- **Strong Emphasis**: Use `60-90%` opacity

### 3. Gradient Usage
- **Decorative Backgrounds**: Use theme gradients
- **Brand Showcase**: Use brand color gradients
- **Special Effects**: Use fruit gradients for fun

## Notes

1. **Accessibility**: Ensure sufficient contrast between background and text colors
2. **Theme Adaptation**: Background colors automatically adapt to light/dark themes
3. **Performance**: Gradients consume more resources than solid colors
4. **Consistency**: Maintain consistent background color usage throughout the application
