cosy-ui provides a series of commonly used icons, using SVG format, and can be directly imported and used.

```astro
import { HomeIcon } from '@coffic/cosy-ui';

<HomeIcon class="cosy:w-8 cosy:h-8" />
```

## Icon List

Here we only list some common icons, for more icons, please check the source code or use the IDE's auto-completion function (the icon component names all have the "Icon" suffix).
