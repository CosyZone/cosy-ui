## Introduction

The AppLayout component is designed for page layouts, providing a complete layout framework that includes sidebar navigation, header, main content area, and footer. It's particularly suitable for documentation sites, blogs, knowledge bases, and other scenarios that require navigation and content display.

## Props

### class:list

Class name list.

<AppLayoutPackage.ClassList />

### class

Custom CSS class name.

<AppLayoutPackage.Class />


### footerConfig

Footer configuration, including site information, social media links, etc. For detailed configuration options, please refer to the [Footer component documentation](/en/manuals/components/footer).

### headerConfig

Header configuration, passed to the AppHeader component. For detailed configuration options, please refer to the [Header component documentation](/en/manuals/components/header).

### loadingDelay

The delay time (in milliseconds) for displaying the loading interface.

<AppLayoutPackage.LoadingDelay />

### mainContentConfig

Main content area configuration.

<AppLayoutPackage.MainContentConfig />

### metaConfig

Metadata configuration, including page title, description, etc.

<AppLayoutPackage.MetaConfig />

### showFooter

Whether to show the footer.

<AppLayoutPackage.ShowFooter />

### showHeader

Whether to show the header.

<AppLayoutPackage.ShowHeader />

### showSidebar

Whether to show the sidebar.

<AppLayoutPackage.ShowSidebar />
