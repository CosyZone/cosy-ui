## Introduction

The AlertDialog component is used to display simple confirmation dialogs with internationalization support and fade-in/fade-out animation effects.

Component Features:

- Supports Chinese and English internationalization
- Features fade-in/fade-out animation effects
- Can be closed by clicking the background or button
- Responsive design that adapts to different screen sizes
- Supports custom dialog ID
- Uses daisyUI styling for consistent design

## API Reference

### Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| message | string | - | The message content displayed in the dialog |
| lang | 'zh-cn' \| 'en' | 'zh-cn' | Language setting that affects button text |
| id | string | 'alert-dialog' | Unique identifier for the dialog |

### Usage Instructions

1. **Show Dialog**: Use the `showAlertDialog(dialogId)` function to display the dialog
2. **Close Dialog**: Click the background overlay or confirm button to close the dialog
3. **Internationalization**: Switch between Chinese and English interfaces using the `lang` property
4. **Custom ID**: Set a unique identifier for the dialog using the `id` property
