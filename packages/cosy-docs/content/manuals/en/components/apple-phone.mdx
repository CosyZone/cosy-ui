# {frontmatter.title}

The ApplePhone component simulates the appearance of an iPhone device, perfect for mobile UI prototyping.

## Examples

<ApplePhoneVuePackage.Basic />

## Props

### backgroundColor
Content area background color, used to customize the background color of the iPhone content area.
<ApplePhoneVuePackage.BackgroundColor />

### debug
Whether to enable debug mode, when enabled, a semi-transparent yellow background will be displayed for layout debugging.
<ApplePhoneVuePackage.Debug />

### height
Window height, supporting multiple predefined size options.
<ApplePhoneVuePackage.Height />

### showFrame
Whether to show the iPhone frame, when set to false, only the content area is displayed.
<ApplePhoneVuePackage.ShowFrame />

### statusBarButtons
Status bar buttons array, used to add custom buttons in the status bar.
<ApplePhoneVuePackage.StatusBarButtons />

### title
Window title, displayed as title text in the status bar.
<ApplePhoneVuePackage.Title />

### withShadow
Whether to show shadow effects, adding dimensionality to the iPhone window.
<ApplePhoneVuePackage.WithShadow />

## Slots

### default
Main content area, used to place application interface content.
<ApplePhoneVuePackage.Slots />
