## Introduction

The SmartIcon component is an intelligent icon selector that automatically chooses the most appropriate icon based on keywords. It supports multiple keywords, prioritizes the first matching icon, and renders a default icon when no match is found. This component is particularly suitable for scenarios that require dynamic icon display based on context.

## Props

### keyword

Keywords, supporting multiple keywords separated by spaces or commas. The component will automatically select the most appropriate icon based on the keywords.

<SmartIcon keyword="user" size="32px" color="#8b5cf6" />

### size

The size of the icon, supports any valid CSS dimension value.

<div style="display: flex; gap: 1rem; align-items: center;">

  <SmartIcon keyword="settings" size="24px" color="#8b5cf6" />
  <SmartIcon keyword="search" size="32px" color="#8b5cf6" />
  <SmartIcon keyword="document" size="48px" color="#8b5cf6" />
</div>

### color

The color of the icon, supports any valid CSS color value.

### class

Custom CSS class name for overriding default styles.

SmartIcon supports multiple keywords separated by spaces or commas:

<SmartIcon keyword="user settings" size="32px" color="#8b5cf6" />

<SmartIcon keyword="code,programming,development" size="32px" color="#8b5cf6" />

## Use Cases

- Dynamic content display, automatically selecting icons based on content type
- Smart search result icon display
- Automated user interface generation
- Content management system dynamic icons
- Context-based icon selection

## Intelligent Matching Logic

1. **Direct Match** - Prioritize exact keyword matches
2. **Contains Match** - If direct match fails, try contains matching
3. **Default Icon** - Use document icon as default when no match is found
4. **Keyword Priority** - When multiple keywords exist, use the first matching icon

## Supported Keyword Categories

- **User Related**: user, person, account, profile, team, member
- **Function Related**: settings, search, edit, delete, save, upload, download
- **Status Related**: success, warning, error, info, complete, in-progress
- **Technology Related**: code, programming, development, database, network, API, framework
- **Content Related**: document, file, image, video, audio, link
- **Time Related**: calendar, clock, history, plan, reminder
