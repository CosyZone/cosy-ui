# {frontmatter.title}

 Picture book layout components.

 - `PictureBookPage` provides a fixed aspect ratio canvas;
 - `PictureBookTextBox` positions by percentage and auto-sizes height by content by default.

## Introduction
Great for illustrated pages, reading booklets, and cover layouts.
