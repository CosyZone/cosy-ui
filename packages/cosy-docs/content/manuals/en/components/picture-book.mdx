# {frontmatter.title}

Picture book layout components. `PictureBookPage` provides a fixed aspect ratio canvas with line-based layout; `PictureBookTextBox` positions by percentage and can set height by number of lines using the parent line height.

## Introduction
Great for illustrated pages, reading booklets, and cover layouts.
