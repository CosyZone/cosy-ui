## Introduction

The SocialIcon component is used to display social media platform icons, supporting various common social media platforms. It provides consistent styling and interaction effects while supporting custom sizes and colors.

## Props

### platform

Social media platform name, supports the following platforms:

- `github` - GitHub platform
- `twitter` - Twitter platform  
- `linkedin` - LinkedIn platform
- `facebook` - Facebook platform

<SocialIcon platform="github" size="32px" color="#333" />

### size

The size of the icon, supports any valid CSS dimension value.

<div style="display: flex; gap: 1rem; align-items: center;">
  <SocialIcon platform="github" size="16px" color="#333" />
  <SocialIcon platform="github" size="24px" color="#333" />
  <SocialIcon platform="github" size="32px" color="#333" />
  <SocialIcon platform="github" size="48px" color="#333" />
</div>

### color

The color of the icon, supports any valid CSS color value.

<div style="display: flex; gap: 1rem; align-items: center;">
  <SocialIcon platform="github" size="24px" color="#333" />
  <SocialIcon platform="github" size="24px" color="#0366d6" />
  <SocialIcon platform="github" size="24px" color="#28a745" />
  <SocialIcon platform="github" size="24px" color="#d73a49" />
</div>

### class

Custom CSS class name for overriding default styles.

<SocialIcon platform="github" size="32px" color="#333" class="custom-social-icon" />

## Use Cases

- Social media link buttons
- Team member social profile display
- Website social sharing functionality
- In-app social login options

## Design Philosophy

1. **Clear and Simple** - Use clear and recognizable icons to represent social media platforms
2. **Consistency** - Maintain unified styling and interaction effects
3. **Customizability** - Support custom sizes, colors, and styles
4. **Accessibility** - Provide appropriate aria labels to ensure screen readers can identify them
