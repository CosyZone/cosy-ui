# Icons Vue

## Introduction

Icons Vue component library provides a complete set of Vue icon components based on SVG icons, supporting customizable size, color, and styles. All icons are carefully designed to ensure clarity and beauty at different sizes.

## Usage

### Basic Usage

```vue
<template>
  <UserIcon />
  <SettingsIcon size="32px" color="blue" />
  <SearchIcon class="custom-icon" />
</template>

<script setup>
import { UserIcon, SettingsIcon, SearchIcon } from '@coffic/cosy-ui/vue';
</script>
```

### Custom Styling

```vue
<template>
  <div>
    <HeartIcon 
      size="48px" 
      color="#ef4444" 
      class="cosy:animate-pulse" 
    />
  </div>
</template>

<script setup>
import { HeartIcon } from '@coffic/cosy-ui/vue';
</script>
```

## Technical Features

- **TypeScript Support** - Complete type definitions
- **Responsive Design** - Support for different screen sizes
- **Accessibility** - WCAG compliant
- **Performance Optimized** - Lightweight SVG icons
- **Theme Support** - Support for dark/light theme switching

## Notes

1. All icons are based on a 24x24 grid design to ensure visual consistency
2. Icon color defaults to `currentColor` and automatically inherits the parent element's text color
3. Icon size supports any valid CSS dimension value, including pixels, em, rem, etc.
4. Custom class names can be passed through the `class` attribute, supporting frameworks like Tailwind CSS
