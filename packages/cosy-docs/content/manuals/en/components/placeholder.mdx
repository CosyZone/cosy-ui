## Introduction

The PlaceHolder component is used to occupy specified space in layouts, similar to SwiftUI's Spacer. You can independently set the width and height of the component, supporting different size options and background color configurations.

## Features

- **Space Occupation** - Occupy specified space in layouts
- **Flexibility** - Support independent control of width and height
- **Consistency** - Maintain consistent size specifications with the design system
- **Responsive** - Maintain good layout effects on different screen sizes
- **Visualization** - Support background color configuration for easy debugging and design
