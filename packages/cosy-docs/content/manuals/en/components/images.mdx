## Introduction

The Images component is used to display a collection of images with multiple layout options and rich styling effects. It can beautifully showcase image collections like the ones you've shown, supporting grid, masonry, carousel, and stack layouts.

## Props

### class

Custom CSS class name for adding additional styles.

### columns

Number of columns for grid layout, effective when layout is grid.

<ImagesPackage.Columns />

### gap

Image spacing, optional values: none, xs, sm, md, lg, xl.

<ImagesPackage.Gap />

### hover

Hover effects, optional values: none, scale, brightness, blur.

<ImagesPackage.Hover />

### images

Array of images, each containing source URL, alt text, and other information.

<ImagesPackage.Images />

### layout

Layout method, optional values: grid, masonry, carousel, stack.

<ImagesPackage.Layouts />

### rounded

Image border radius, optional values: none, sm, md, lg, xl, 2xl, 3xl, full.

<ImagesPackage.Rounded />

### shadow

Image shadow, optional values: none, sm, md, lg, xl, 2xl.

<ImagesPackage.Shadow />
