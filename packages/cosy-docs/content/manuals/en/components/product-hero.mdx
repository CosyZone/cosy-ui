## Introduction

The ProductHero component is a container component for ProductHeroItem, designed to display detailed information about multiple products. It features an Apple-style large image display design with alternating left-right layouts, providing an elegant product showcase experience. Perfect for product showcase pages, product detail pages, and similar scenarios.

Component Features:

- Large Image Display - Highlights product images with high-quality large image presentation
- Alternating Layout - Automatic left-right alternating layout for enhanced visual hierarchy
- Information Hierarchy - Clear information structure: labels, titles, descriptions, prices, buttons
- Apple Style - Adopts Apple's clean and elegant design philosophy
- Responsive Design - Excellent display across different devices

### products

Array of product data, each product contains the following properties:

- `name`: Product name
- `description`: Product description
- `image`: Product image
- `price`: Product price
- `isNew`: Whether it's a new product (optional)
