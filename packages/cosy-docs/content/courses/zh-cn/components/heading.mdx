Heading 组件用于创建各级标题，提供一致的排版样式和灵活的定制选项。该组件适用于需要清晰层次结构的内容页面，如文档、博客和产品介绍等。

## 基础用法

最基本的标题用法，支持 1-6 级标题：

<ComponentPackage.ComponentContainers.Basic />

## 自定义样式

通过 `color` 和 `underline` 属性自定义标题样式：

<ComponentPackage.ComponentContainers.CustomStyle />

## 对齐方式

通过 `align` 属性控制标题的对齐方式：

<ComponentPackage.ComponentContainers.Align />

## 带锚点的标题

通过 `id` 和 `anchor` 属性创建带锚点的标题：

<ComponentPackage.ComponentContainers.Anchor />

## Props

| 属性         | 说明                   | 类型                                              | 默认值    |
| ------------ | ---------------------- | ------------------------------------------------- | --------- |
| level        | 标题级别               | 1 \| 2 \| 3 \| 4 \| 5 \| 6                      | 2         |
| id           | 标题的 ID              | string                                            | -         |
| anchor       | 是否显示锚点链接图标   | boolean                                           | false     |
| underline    | 是否显示下划线         | boolean                                           | false     |
| align        | 文本对齐方式           | 'left' \| 'center' \| 'right'                    | 'left'    |
| color        | 标题颜色               | 'default'\|'primary'\|'secondary'\|'accent'\|'muted' | 'default' |
| class        | 自定义 CSS 类名        | string                                            | -         |

## 插槽

- `default`：标题内容
