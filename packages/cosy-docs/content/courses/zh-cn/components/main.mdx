Main 组件是页面的主要内容区域，使用 Container 组件提供响应式布局。适合作为页面主体，自动处理不同屏幕尺寸下的布局和内边距。

## 基础用法

<ComponentPackage.ComponentContainers.Basic />

## 自定义容器尺寸

<ComponentPackage.ComponentContainers.CustomSize />

<ComponentPackage.ComponentContainers.CustomPadding />
