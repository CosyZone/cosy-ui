## Introduction

The PlaceHolder component is used to occupy specified space in layouts, similar to SwiftUI's Spacer. You can independently set the width and height of the component, supporting different size options and background color configurations.

## Features

- **Space Occupation** - Occupy specified space in layouts
- **Flexibility** - Support independent control of width and height
- **Consistency** - Maintain consistent size specifications with the design system
- **Responsive** - Maintain good layout effects on different screen sizes
- **Visualization** - Support background color configuration for easy debugging and design

## API Reference

### Props

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `width` | `'none' \| 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl' \| '5xl' \| '6xl' \| 'full'` | `'md'` | Placeholder width |
| `height` | `'none' \| 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| '2xl' \| '3xl' \| '4xl' \| '5xl' \| '6xl' \| 'full'` | `'md'` | Placeholder height |
| `padding` | `'none' \| 'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'none'` | Internal spacing size |
| `background` | `string` | - | Background color class name, such as 'base-200', 'primary', 'secondary' |
| `class` | `string` | - | Custom CSS class name |

## Use Cases

### 1. Layout Placeholder

Use placeholders in layouts to occupy specified space:

```astro
<div class="cosy:bg-base-200">
  <PlaceHolder width="lg" height="md" padding="md" background="base-300">
    <p>This content has appropriate width, height, padding, and background color</p>
  </PlaceHolder>
</div>
```

### 2. Card Layout

Use PlaceHolder in card components to create content areas:

```astro
<div class="cosy:bg-base-100 cosy:rounded-lg cosy:shadow">
  <PlaceHolder width="md" height="lg" padding="lg" background="base-200">
    <h3>Card Title</h3>
    <p>Card content</p>
  </PlaceHolder>
</div>
```

### 3. Debug Layout

Use background colors to visualize layout structure:

```astro
<PlaceHolder width="xl" height="lg" padding="lg" background="primary" class="cosy:text-white">
  <p>This area is used for debugging layout, background color helps visualize space occupation</p>
</PlaceHolder>
```

### 4. Responsive Layout

Combine with responsive class names:

```astro
<PlaceHolder 
  width="lg" 
  height="md" 
  padding="md" 
  background="base-200"
  class="cosy:lg:width-xl cosy:lg:height-lg cosy:lg:padding-lg"
>
  <p>Responsive placeholder, larger dimensions on desktop screens</p>
</PlaceHolder>
```

### 5. Fill Container

Use the `full` option to make the placeholder fill the entire container:

```astro
<div class="cosy:h-64 cosy:bg-base-200">
  <PlaceHolder width="full" height="full" padding="md" background="base-300">
    <p>Placeholder that fills the entire container</p>
  </PlaceHolder>
</div>
```
