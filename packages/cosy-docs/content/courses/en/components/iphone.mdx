## Props

| Name            | Description                | Type      | Default    |
| --------------- | ------------------------- | --------- | ---------- |
| height          | Window height             | string    | h-96       |
| title           | Window title              | string    | ''         |
| statusBarButtons| Status bar buttons array  | array     | []         |
| withShadow      | Show shadow effect        | boolean   | true       |
| showFrame       | Show iPhone frame         | boolean   | true       |
| backgroundColor | Content area background   | string    | ''         |

## Slots

- `default`: Main content area
